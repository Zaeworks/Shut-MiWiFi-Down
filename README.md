# Shut-MiWiFi-Down

一键关掉小米路由器的PPPoE连接

给那些需要这种特殊场景的人用.

- 你的寝室断电
- 你的路由器如果突然断电, 再次连上PPPoE需要等五六分钟
- 你在断电前突然想起了这件事, 但小米路由器的管理页面非常卡, 登陆非常慢.

那么你需要这个.

## 使用

[下载程序](https://github.com/Zaeworks/Shut-MiWiFi-Down/releases)

初次运行程序, 会在目录下生成一个同名xml文件, 在里面输入登录密码和路由器管理页面IP.

之后运行程序便会自动登陆. 登录成功后, 敲一下按键就可以一键断掉PPPoE连接.
